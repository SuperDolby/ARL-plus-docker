# ARL-plus-docker
基于(https://github.com/TophantTechnology/ARL/)

# 使用方式
进入之前的arl/docker目录
```docker-compose down```

然后在该目录下
```docker-compose up -d```

修改OneForAll配置文件

使用中央数据库需要外联站点，私聊我加ip白名单
